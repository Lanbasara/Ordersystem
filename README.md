# Ordersystem
Online recruitment app with real-time chat function

# Features
* Technology stack ： React+Express+Mongodb+websocket
* Simple and beautiful style ： Antd Mobile
* Real-time Chating : websocket
* User Authentication
* Dual MD5 password encryption
